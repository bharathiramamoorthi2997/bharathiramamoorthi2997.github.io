Services

A bootcamp is a program where aspiring learners can pick up lot of momentum by learning a lot in a very short time. Through Code Intensive, a 3 day residential camp, participants will tremendously benefit by gaining solid programming skills. Code Intensive will be rigorous and hence it is for the passionate learners who can pull through 3 intense days and nights to emerge successful. Well, this is how superheroes are made :)

Full Stack Web and App Development

You will hone your understanding of how the web works, develop complex relational databases used to store applications data, secure and configure your own Linux-based servers, and build complete web applications using Python, HTML5, CSS, Bootstrap, JavaScript and SQL. You will also learn modern Frameworks and technologies to build mobile apps from web apps using Ionic.

Product Building and Launching

The final module of this bootcamp is product building. The final project is your opportunity to take your new-found savvy with programming and develop your very own piece of software application. You would work with a team, simulate real life software development lifecycle and build a product from an idea to a fully working alpha release.

Computational Thinking and Algorithms

Computational Thinking (CT) is a problem solving process that includes these capabilities: 1. logically ordering & analyzing data, 2. creating solutions using well-designed algorithms & dispositions 3. the ability to confidently deal with complexity and open-ended problems.

Diverse CS Languages and Constructs

This module gives a broad and robust understanding of computer science and the art of programming. Topics include abstraction, constructs, data structures, encapsulation, resource management, security, software engineering, and web development. Languages include C, Python, Java, SQL, and JavaScript plus CSS and HTML.
